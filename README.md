# Ethical Hacking Summer Training

This repository documents my four-week Ethical Hacking Summer Training.

## Topics Covered

- Footprinting and Reconnaissance
- Network Scanning
- Enumeration
- Vulnerability Assessment
- System Hacking
- Password Cracking
- Malware
- Sniffing
- Social Engineering
- DoS and DDoS
- Session Hijacking
- SQL Injection

## Tools Used

- Kali Linux
- VirtualBox
- Nmap
- Wireshark
- Burp Suite
- Recon-ng
- OpenVAS

## Learning Outcomes

- Learned ethical hacking methodology
- Performed network reconnaissance
- Conducted port scanning
- Captured and analyzed network traffic
- Documented practical lab exercises
